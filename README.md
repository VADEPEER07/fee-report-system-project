# Fee Report System

Java desktop application using JDBC and MySQL for student fee management.

## Features
- Login authentication
- Student record management
- Payment tracking
- Receipt generation

## Tech Stack
- Java (Swing)
- MySQL
- JDBC

## How to Run
1. Import `fee_report_db.sql` into your MySQL.
2. Update your JDBC credentials.
3. Run `Main.java`.
